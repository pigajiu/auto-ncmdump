## 功能 Use

可实现后台挂起无感知地全自动ncm to mp3。  
运行后，在网易云音乐客户端上选曲下载，下载成功即转换成功。  
或者，先下载完成后，再运行本工具，依次转换成mp3。

## 使用 Usage

无需 Python 环境，仅支持 Windows 10操作系统。  
下载 [auto-ncmdump.exe](https://github.com/pigajiu/auto-ncmdump/releases/download/1.1/autodump.exe "auto-ncmdump.exe")  后运行，按照操作提示使用。
